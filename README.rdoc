= Queueable-web

Queueable is a test suite for measuring IPC performance and tolerance
to concurrency. Queueable-web simply stores and displays the results of
each test run. Live site is at https://queueable.herokuapp.com .

See https://github.com/adamonduty/queueable for more information.
